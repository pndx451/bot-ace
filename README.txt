ACE BYPASS BOT V2

INSTALL:
1. npm install
2. copy .env.example .env
3. Put your Discord bot TOKEN in .env
4. node index.js

DISCORD DEVELOPER PORTAL:
Scopes:
- bot
- applications.commands

Bot Permissions:
- Administrator

Privileged Gateway Intents:
- Presence Intent
- Server Members Intent
- Message Content Intent

IMPORTANT:
Only these IDs can run commands:
- 1484280198362042461
- 1217889927703040060

Commands can be used from any channel.
Most commands include an optional channel field so you can choose where the embed/panel is sent.

For exact visual hiding of commands in Discord, use:
Server Settings > Integrations > ACE BYPASS > Commands
and restrict commands manually to your staff/users.
